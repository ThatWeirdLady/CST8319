# Solitaire

## Description

A Solitaire game that runs in browser.  

You can interact with the live demo here: https://thatweirdlady.github.io/Solitaire/ 

**Features:**
- Draw One or Three
- Klondike
- Vegas

## Technologies Used

- Typescript
- Webpack 5
- ES Lint
- Prettier

### Requirements to run locally:

Download essential libraries
```sh 
npm ci
```

Run program
```sh
npm run start
```
